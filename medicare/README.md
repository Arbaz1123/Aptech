# MediCare – Healthcare / Doctor Website

A modern, responsive healthcare website template built with HTML, CSS, Bootstrap 5, jQuery, AOS, and GSAP. Includes hero, services, doctors, appointment form, contact map, and stats counters.

## Quick start

1. Open `index.html` in your browser.
2. Ensure internet access for CDN assets (Bootstrap, jQuery, AOS, GSAP, Font Awesome, Google Maps).

## Features

- Hero section with CTA button and image
- Services grid with icons
- Doctors profiles with social links
- Appointment form with validation
- Contact info with Google Maps embed
- Stats counters with animations
- Smooth scrolling, navbar shrink on scroll, back-to-top button
- AOS scroll animations and subtle GSAP transitions

## Customize

- Colors and spacing: `assets/css/styles.css` (CSS variables at the top)
- Images: replace URLs or add files to `assets/img/` and update `index.html`
- Services/Doctors: edit the corresponding cards in `index.html`
- Form handling: replace the simulated submit in `assets/js/main.js` with your backend endpoint
